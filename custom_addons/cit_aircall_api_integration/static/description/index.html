Aircall Api Integration

The latest release of the Aircall Base Module introduces several key functionalities, including the ability to fetch the total number of users in an account and add user-based lognotes. Contact management is enhanced with support for multiple contacts sharing the same phone number and fields for both mobile and phone numbers. Additionally, users can now selectively add lognotes to specific contacts, and the creation of unknown contacts for inbound calls is configurable. These updates significantly improve user experience and operational efficiency in Aircall.



How It is Works?

This module allows you to make and receive calls directly within Odoo, making it easy to connect with your customers.



Below is a list of the features included in the module.

FEATURES

Handle Multiple Contacts with same phone Number.

  • Able to handle multiple contacts sharing the same phone number. In case you have multiple contact with the same phone number, Insight cards will display a link to the matching contacts and when clicking on insight cards you will be taken to a page with a list of contacts with that number. There you can choose where to add the log note or activity after the call is ended.

Configurable Unknown Contact Creation.

  • Decide what happens when a contact is not found in Odoo during inbound calls. You can decide if you want to create an unknown contact to get you started. It will only have the phone number.

Find Calls Summary(Call History).

  • List of calls will be added to the contact under Aircall details tab. There you can find a list of calls with the user, contact name, numbers and link to recordings. If you click on those calls a menu with more data will show.

  • You can also find all call details in the Aircall menu.

Dialing Aircall Contact Directly from Odoo.

  • With the Click to Dial feature, you can effortlessly make calls directly from Odoo. By simply clicking on the call button, you can initiate calls without the need for manual dialing. This functionality enhances productivity and streamlines your communication process, allowing you to connect with customers, partners, and team members quickly and efficiently from within the Odoo interface.

Sync the Updated Contact.

  • When a contact is updated in Odoo, the changes will also be reflected in Aircall. This synchronization ensures that your contact information remains consistent across both platforms. Please note that in some cases, it may take up to 5 minutes for the updates to be processed and appear in Aircall. This delay allows for a smooth and reliable update process, ensuring that all contact details are accurately maintained.

Receive Incoming calls in Odoo.

  • When an incoming call is received, Odoo provides a pop-up notification, enabling you to view caller information and access relevant records or customer details instantly.

Redirect from Insight Card to Particulate contact.

  • Easily manage your inbound calls with the Access Contacts from the Insight Card feature. When a contact calls, you can access their information directly from the insight card. By clicking on "Select Contact," you'll see a list of related contacts with matching phone or mobile numbers. If there is only one contact related to that number, the contact name will be displayed directly in the insight card. This allows for quick identification and seamless handling of calls, ensuring you have all necessary information at your fingertips.

Assign Call to any User in Odoo.

  • With the Assign Call to Any User feature, you can easily reassign calls from one user to another within Odoo. This functionality allows for efficient call management and ensures that the right person handles each call. Whether you need to delegate tasks or reallocate responsibilities, this feature provides flexibility and improves workflow by enabling smooth call assignment between users.

Fetch Aircall User Count.

  • After completing the configuration in settings, you can easily identify the number of Aircall users allowed to add log notes in contacts. This feature provides a clear overview of user permissions, ensuring that only authorized users can contribute log notes. It streamlines the management of user access and maintains the integrity of your contact records.

During Call, Add notes or Comments, Tags.

  • During calls, Aircall allows you to add notes, comments, and tags specific to each call. This information is then automatically added to the log note when the log note is created in the contact record. This feature ensures that all relevant details and annotations from the call are captured and easily accessible within the contact's history, enhancing documentation and follow-up efficiency.

Real-time Contact Number Sync.

  • Ensure seamless communication with the Real-time Contact Number Sync feature, which synchronizes contact numbers between Aircall and Odoo in real-time. Any updates made to contact numbers in Aircall are instantly reflected in Odoo, and vice versa. This ensures that your contact information is always up-to-date across both platforms, facilitating smooth and efficient interactions. Note : It take 1 minutes to sync contact with aircall

Get Call Recordings, metrics and log Activities easily in odoo.

  • After a call ends, a summary of the call is automatically added to the contact object in Odoo. Please note that it may take up to 1 minute for the recording and voicemails to be processed. The summary includes essential details such as Call ID, Number, Call Qualification, Aircall User, Call Duration, Waiting Time, Talk Time, Tags, and Comments.
  • If there is only one contact associated with the phone number, the summary will be automatically logged into that contact's record. If multiple contacts are associated with the number, you will need to use the insight cards to select the appropriate contact for logging the information. Otherwise, the summary will not be logged.
FREQUENTLY ASK QUESTIONS
How to used fetch numbers button in AirCall configuration?

Fetching all aircall numbers in odoo using the fetch numbers button in the configuration setting after fetching number logs and recording functionality is working for aircall.

How to change number in AirCall?

AirCall number is not changeable because it is set when you are create your AirCall account.

How to change / add number in dial?

If you want to change the number, the option is available under the dialpad on the AirCall dialpad, and if you want to add a number, go to the AirCall account and click on the Team & User tab and add a user.

Where we can see odoo contacts in AirCall?

Go to the AirCall dialpad and click on the people menu, then select the contacts tab, and you can see the Odoo contacts.
